# Tips
1. 有系统地设计文件目录，不要随便到处保存文件以至以后不知道放哪里了， 或找到文件也不知道为何物。
2. 养成一个做记录的习惯。尤其是发现问题的时候，把错误信息和引发状况以及解决方法记录清楚，同时最后归类及定期整理。
3. 如果看在网络上看到任何好文章，可以定好题目，归类存档.
4. 谨记: 不同的环境下，解决问题的方法有很多种，只要行的通，就是好方法！
